# Codex PR Lab (90 minutes)

## Goal

Enable engineers to ship one high-quality PR with evidence artifacts using Codex harness workflows.

## Agenda

1. 10 min: Harness model and review policy
2. 15 min: Command interface walkthrough
3. 45 min: Hands-on lab (issue -> change -> evidence -> PR draft)
4. 10 min: Debrief on failures and missing capabilities
5. 10 min: Adoption next steps

## Lab Tasks

1. Run `pnpm lint:docs` and fix one intentional docs issue.
2. Run `pnpm lint:architecture` and inspect layer constraints.
3. Run `pnpm verify:ui` and `pnpm verify:obs`.
4. Draft PR body with required review evidence fields.

## Success Rubric

- Participant produced a valid PR body with evidence and rollback.
- Participant can explain one guardrail and one failure mode.
- Participant can identify where to encode a repeated human review comment.
