# Security

Owner: Nick
Last Verified: 2026-03-05
Status: Active

## Public-Safe Rules

1. No private company data in commits.
2. No secrets in repo, scripts, or docs.
3. Demo and test data must be synthetic only.

## Agent Safety Controls

1. Use deterministic scripts for sensitive checks.
2. Require explicit evidence for runtime-impacting changes.
3. Escalate policy or architecture exceptions to human review.

For vulnerability disclosure process, see [../SECURITY.md](../SECURITY.md).
